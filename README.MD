## Getting Started

1. Clone this repository

   ```bash
   git clone https://github.com/junaediakbar/laundry-api.git
   ```

2. Copy .env.example to .env

3. Install dependencies

   ```bash
   npm install
   yarn
   ```

4. Run it on your machine

   ```bash
   npm run dev
   yarn dev
   ```
